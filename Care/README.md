# Care App

A comprehensive family collaboration platform for caring for aging parents.

## Features

- **Schedule Management**: Coordinate family visits and appointments
- **Health Tracking**: Log vital signs and health metrics
- **Medication Management**: Track medication lists and schedules
- **Medical History**: Document hospital visits and medical events
- **Care Notes**: Share observations and updates between family members
- **Task Assignment**: Distribute caregiving responsibilities
- **Document Storage**: Store important medical and legal documents
- **Emergency Contacts**: Quick access to important contacts

## Project Structure

- `ios/` - iOS native app
- `android/` - Android native app
- `web/` - Web application
- `backend/` - Server and API
- `shared/` - Cross-platform shared code and models
